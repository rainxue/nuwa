# 概览
nfs模块的主要功能是提供对对象存储服务（Object Storage Service, OSS）的访问接口。它支持常见的对象存储操作，如上传、下载、删除和列出对象等。
用于与各种对象存储服务（如AWS S3、阿里云OSS、腾讯云COS等）进行交互，简化了对象存储的使用流程。

# 主要特性包括
- 支持多种对象存储服务提供商
- 提供统一的API接口，为内部业务服务提供统一的对象存储访问方式
- 支持对象的基本操作，如上传、下载、删除和列出对象
- 支持旁路上传（Presigned URL）功能，允许客户端直接与对象存储服务交互
- 支持多租户，对租户的存储进行隔离，并可独立计算和限制存储空间的大小
